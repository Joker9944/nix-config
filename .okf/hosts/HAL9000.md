# Hardware

Desktop tower, x86_64-linux, Nvidia GPU. Three DisplayPort monitors:

| Output | Mode | Position | Role |
|---|---|---|---|
| DP-1 | 1920x1080@60 | 0,0 | Left |
| DP-2 | 2560x1440@144 | 1920,0 | Center (primary) |
| DP-3 | 1920x1080@60 | 4480,0 | Right |

Monitor and workspace bindings live in `users/joker9944/hosts/HAL9000/default.nix#wayland.windowManager.hyprland.settings`. The primary is set at hyprland start via an `xrandr --output DP-2 --primary` call built with `custom.lib.mkLuaCall`.

# Host record & mixin selection

Host record: `flake.nix#nixosConfigurations`. Enabled NixOS mixins: `hosts/HAL9000/mixins.nix`. Per-host quirks (red limine boot branding, `system.stateVersion = "24.05"`, regreet window rule pinned to DP-2): `hosts/HAL9000/default.nix`.

Home-manager side: `users/joker9944/hosts/HAL9000/default.nix`. Host-specific additions worth calling out:

* Gaming stack: Prism Launcher, Lutris, Steam autostart entry.
* Multiple Librewolf profiles: Work / Homework.
* `programs.yas.config.gpu = true;` and the waybar GPU widget — HAL9000-only because both depend on the discrete GPU.
* GNOME touchpad disabled — guarded by `mixins.desktopEnvironment.gnome.enable`, so it's inert under hyprland.
